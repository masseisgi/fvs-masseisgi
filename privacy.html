Política de Privacidade — MGI PRO

Última atualização: 23 de abril de 2026
Resumo: O MGI PRO é um aplicativo de uso profissional desenvolvido pela Massei Gestão Integrada para auditoria ISO 9001, PBQP-H, verificação de serviços e diário de obra. Os dados do usuário são armazenados localmente no próprio dispositivo e, quando configurada, a API da Anthropic (Claude) é utilizada para geração de ações corretivas. O aplicativo não coleta dados para fins de marketing nem compartilha informações com terceiros além dos provedores de serviço diretamente envolvidos no funcionamento.

1. Quem somos

Massei Gestão Integrada
E-mail de contato: contato@masseisgi.com.br
Site: masseisgi.github.io

Somos os responsáveis pelo tratamento dos dados coletados pelo aplicativo MGI PRO, conforme a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

2. Dados coletados pelo aplicativo

O MGI PRO coleta apenas os dados estritamente necessários ao seu funcionamento:

2.1 Dados inseridos pelo usuário

  • Informações de identificação da auditoria (empresa, processo, auditor, obra)
  • Respostas dos checklists de auditoria e FVS
  • Observações e evidências textuais
  • Fotografias tiradas via câmera do dispositivo ou selecionadas da galeria
  • Assinaturas digitais desenhadas na tela
  • Dados do Diário de Obra (condições climáticas, mão de obra, equipamentos, atividades)

2.2 Configurações do dispositivo

  • API Key da Anthropic (inserida pelo usuário, armazenada apenas no dispositivo)
  • Preferências de interface

2.3 O que NÃO coletamos

  • ❌ Localização GPS
  • ❌ Lista de contatos
  • ❌ Dados de navegação ou cookies publicitários
  • ❌ Identificadores de dispositivo para rastreamento
  • ❌ Dados biométricos além das assinaturas explicitamente desenhadas

3. Como os dados são armazenados

3.1 Armazenamento local

Todos os dados inseridos no aplicativo ficam armazenados exclusivamente no dispositivo do usuário, através do mecanismo localStorage do navegador (WebView). Os dados não são enviados para servidores da Massei, tampouco para nuvem de terceiros, exceto quando o usuário utiliza a função de geração de ações corretivas por Inteligência Artificial (ver item 4).

3.2 Permanência dos dados

Os dados permanecem no dispositivo até que o usuário:

  • Exclua manualmente um registro pelo próprio aplicativo
  • Limpe o armazenamento do aplicativo nas configurações do Android
  • Desinstale o aplicativo

4. Serviços de Inteligência Artificial

O MGI PRO utiliza a API da Anthropic (Claude) para gerar ações corretivas 5W2H e análises de produtividade do Diário de Obra. Essa funcionalidade só é ativada quando o usuário configura sua API Key pessoal nas configurações do app.

Quando ativada:

  • As descrições textuais das Não Conformidades são enviadas à Anthropic para processamento
  • Os dados do Diário de Obra consolidado são enviados para geração de análise
  • As fotografias não são enviadas à IA
  • A Anthropic processa os dados conforme sua própria política de privacidade: anthropic.com/privacy

A API Key da Anthropic fica armazenada somente no dispositivo do usuário e nunca é compartilhada com a Massei.

5. Permissões solicitadas pelo aplicativo

  • Câmera: para registro fotográfico de evidências (opcional, pode ser negada)
  • Arquivos/Fotos: para importar imagens da galeria (opcional)
  • Internet: apenas quando a função de IA é utilizada

6. Compartilhamento de dados

A Massei Gestão Integrada não compartilha, não vende e não aluga dados dos usuários. Os únicos compartilhamentos que ocorrem são:

  • Anthropic — quando o usuário utiliza a função de IA (conforme item 4)
  • Google Play Services — dados técnicos de crash e diagnóstico do Android, conforme política do Google

7. Segurança

  • Conexões com a API da Anthropic utilizam HTTPS/TLS
  • Dados armazenados localmente usam a proteção nativa do Android
  • O app não exige login, reduzindo a superfície de coleta de dados
  • A API Key do usuário nunca sai do dispositivo, exceto em requisições diretas à Anthropic

8. Direitos do titular dos dados (LGPD)

Como usuário do MGI PRO, você tem direito a:

  • Acessar seus dados (estão no seu próprio dispositivo)
  • Retificar ou atualizar dados
  • Excluir seus dados (a qualquer momento, dentro do app)
  • Portar seus dados (via exportação em PDF)
  • Revogar consentimento (basta desinstalar o aplicativo)
  • Solicitar informações sobre o tratamento de dados pela Massei

Para exercer qualquer direito, entre em contato: contato@masseisgi.com.br

9. Crianças e adolescentes

O MGI PRO é destinado exclusivamente a profissionais adultos que atuam em engenharia, gestão da qualidade e construção civil. Não é indicado para uso por menores de 18 anos e não coleta dados de crianças ou adolescentes.

10. Alterações nesta Política

Esta política pode ser atualizada periodicamente. A versão mais recente está sempre disponível em masseisgi.github.io/fvs-masseisgi/privacy.html. A data de atualização fica registrada no topo do documento.
